How to use NewtonBot

Although he can do some pretty hard math problems, NewtonBot isn't the smartest of robots , so you kinda have to talk down to him in a pretty simple manner. He takes simple commands like the ones you see below. All you have to do is give him a command and something to work with and boom! He'll do it! But, if for whatever reason he doesn't, then he'll let you know he didn't understand, and he'll remind you how to talk to him. If you have any further questions on how to get the most out of your conversation with him, don't hesitate to DM or @ me on Twitter.

A simple command given to NewtonBot looks like this: /operation expression. Where operation is one of the verbs found in the table below and expression is a number or math expression that you want NewtonBot to perform the operation on.

Note: The vertical line in some commands is not a lowercase 'L' or uppercase 'I'. It is a vertical bar.

Operation Command Description Example Result
Simplify /simplify "" Algebraically simplify an expression. /simplify x + x 2x
Factor /factor "" Find the factors of a polynomial expression. /factor x^2 + 2x x (x + 2)
Differentiate /derive "" Find the first derivative of an expression. /derive x^2+2x 2 x + 2
Integrate /integrate "" Find the indefinite integral of an expression. /integrate x^2+2x 1/3 x^3 + x^2 + C
Solve /zeroes "" Solve a polynomial expression. /zeroes x^2+2x [-2, 0]
Find Tangent Line /tangent c|f(x) Find the line tangent to the given function at c. /tangent 2|x^3 12 x + -16
Calculate Area Under Curve /area a:b|f(x) Find the area under the given function between a and b. /area 2:4|x^3 60
Cosine /cos "" Calculate cosine of a value. /cos pi -1
Sine /sin "" Calculate sine of a value. /sin pi / 2 1
Tangent /tan "" Calculate tangent of a value. /tan pi / 4 1
Inverse Cosine /arccos "" Calculate the arccosine of a value. /arccos 1 0
Inverse Sine /arcsin "" Calculate the arcsine of a value. /arcsin 1 1 / 2 pi
Inverse Tangent /arctan "" Calculate the arctangent of a value. /arctan 0 0
Absolute Value /abs "" Calculate the absolute value of a number. /abs -1 1
Logarithm /log a|b Calculate the logarithm of b with base a. /log/2|8 3
